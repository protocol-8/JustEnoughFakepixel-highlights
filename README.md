# JustEnoughFakepixel
A Quality-of-life mod made for fakepixel skyblock.

[Download on modrinth](https://modrinth.com/mod/justenoughfakepixel) - Currently under review.

# Features

## General
* Hide damage splashes
* Roman Numerals
* SkyblockID
* Disable enchant glint

## Diana
* Diana Tracker
* Party commands for diana
 !help

## Misc
* Item Stack Utils
* Brewing Helper
* Search bar
* No swap animation
* Show own nametag
* Disable entity fire

## Custom Scoreboard
* Scoreboard additions like Banking data , Power Selected
* Scoreboard opacity , rounded edges , position control
  
## Waypoints
* Ordered waypoints

## Dungeons
* Blood mob highlight
* Phase timers
* !pb <floor> party command

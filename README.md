# JustEnoughFakepixel
A Quality-of-life mod made for Fakepixel Skyblock.

[Download on Modrinth](https://modrinth.com/mod/justenoughfakepixel) - Currently under review.

# Features

## General
* Hide damage splashes
* Roman Numerals
* Skyblock Item ID tooltip
* Disable enchant glint
* Prevent cursor reset

## Diana
* Diana tracker
* Diana mob health overlay
* Party commands (`!help`)

## Misc
* Item stack utils
* Brewing helper
* Search bar
* No swap animation
* Show own nametag
* Disable entity fire
* Current pet overlay
* Skill XP display

## Scoreboard
* Custom scoreboard with bank balance and selected power
* Opacity, rounded corners, and position control

## Waypoints
* Ordered waypoints with group management

## Dungeons
* Blood mob highlight
* Phase timers
* `!pb <floor>` party command

## Mining
* Fetchur overlay

# IMPORTANT
This mod collects your **Minecraft username** and **installed mod list** on server join.
This data is used solely for analytics and tracking issues related to compatibility.
No other data is collected. No passwords, files, or server IPs are ever accessed.
